Gears

Introduction

Welcome to Gears, a small program to generate some gears in the SVG format. Primary goal of this program was to generate gears for the Hobbymat MD65 Lathe, hence the custom bearing geometry in the center of the gear.

Hopefully, users will be able to generate all kind of gears.

Usage

From the command line

> python gears.py -h
> python gears.py

Further processing

It can be possible to generate stl files within blender in order to use 3D printing. When importing the SVG in blender, the user has to convert the curves to mesh, and then solidify these meshes. Unfortunately, these meshes can be buggy. The user will recognize when something goes wrong.

In this case, the user has to "clean" the mesh from unwanted nodes, especially the spurious one on the teeths of the gear. After having applied some solidify/boolean modifiers, and applied (blender "apply" modifiers) them, then the unwanted items fot the stl export must be deleted.

Finally, the exported stl file can be imported in a 3D printing slicer.
